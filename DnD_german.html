Schaden ++
 
  • Primärattribut : Geschicklichkeit
  • Trefferwürfel : 1W4 pro Stufe
  • Waffe : jede
  • Rüstung : Leder, keine Schilde
  • Spezielle Fähigkeiten :
    • Hinterhältiger Angriff : +4 Bonus auf TW und Schaden x2 wenn erfolgreicher Treffer unbemerkt von hinten
    • Schlösser öffnen : Darf nur einmal pro Schloss versucht werden, außer der Dieb erreicht eine höhere Erfahrungsstufe.
    • Fallen finden/ entschärfen : Darf auch nur einmal pro Falle angewendet werden.
    • Taschendiebstahl : Wird durch den SL eine doppelt so hohe Prozentzahl erwürfelt, wird Dieb gefasst. Der SL bestimmt die Reaktion des Opfers, i.d.R nicht freundlich.
    • Schleichen : Der Dieb ist immer der Meinung, das er erfolgreich ist. Doch der SL weiss es besser!
    • Wände erklimmen : Es wird für jeweils 100e gewürfelt. Bei einem Misserfolg erleidet der Dieb für jeweils 10e 1W6 Sturzschaden.
    • Im Schatten verstecken : Kann nur ausgeführt werden, wenn der Dieb sich still verhält (nicht angreifen bzw. bewegen). Nur der SL weiss, ob der Dieb erfolgreich war.
    • Geräusche hören : Damit ist das Lauschen an Türen und das Hören herannahender Schritte gemeint. Kann nicht im Kampf ausgeführt werden. (zuviele Geräusche!)
